# Monte Carlo Simulation Engine

A Python-based Monte Carlo Simulation project that demonstrates probability concepts and stock price forecasting using real-world data.

## Features

- Coin Toss Simulation
- Dice Roll Simulation
- Stock Price Simulation (Monte Carlo)
- Risk Analysis (Mean, Min, Max, Probability of Loss)
- Data Visualization using Matplotlib

## Technologies Used

- Python
- NumPy
- Matplotlib
- yFinance

## How to Run

1. Clone the repository:
   git clone https://github.com/mrinalishh/Monte-Carlo-Simulation.git

2. Navigate to project folder:
   cd Monte-Carlo-Simulation

3. Install dependencies:
   pip install -r requirements.txt

4. Run the program:
   python main.py

## Sample Output

- Stock price simulation graph
- Final price distribution histogram
- Risk analysis metrics
